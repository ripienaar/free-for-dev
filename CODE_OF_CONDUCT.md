# Code of Conduct

This project adheres to a standard Code of Conduct to ensure a welcoming and respectful community.

Expected behavior
- Be respectful in communications.
- Refrain from harassment, discrimination, or abusive behavior.

Reporting
- If you experience or witness unacceptable behavior, please report it by opening a private issue addressed to the maintainers or using the repository's issue tracker.

For a complete and formal policy, see the Contributor Covenant: https://www.contributor-covenant.org/
